# Проект: Место

### Место

Сервис "Место" - интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.

В работе используются технологии:

* Язык HTML и CSS
* Вёрстка с помощью Grid
* Файловая методология БЭМ
* Базовый JS
